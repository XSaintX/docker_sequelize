# Building REST APIs with Node.js and Express

## Project description
I made this project to improve more my skills as backend and put in practice new concepts.

## Tech Stack
• Postgres and MySQL <br />
• Sequelize <br />
• Docker <br />
• PgAdmin <br />

## Architecture and Concepts Learned
• Security practices: Middleware and error handling <br />
• Working with databases from Node.js <br />
• REST API development with Express <br />
• Environment variables and config management <br />
• Migrations <br />
• Database and ORM techniques <br />
• Relational data modeling with Sequelize <br />

## How to run
• docker-compose up -d postgres <br />
• docker-compose up -d pgadmin <br />
• npm run dev <br />
